VESTIBULANDO — ORIGINAL COM O CHAT ATUAL
=========================================

Este pacote contém:

- Todas as telas, menus, páginas, funcionalidades e o design do projeto
  original enviado em VESTIBULANDO-GITHUB-CLOUDFLARE-CORRIGIDO(1).zip.
- O chat preservado na versão mais recente.
- As regras e os índices necessários ao funcionamento atual do chat.
- Nenhuma tela do painel moderno R11/R12/R13.

COMO PUBLICAR
-------------

1. Extraia o ZIP em uma pasta nova.
2. Abra a pasta VESTIBULANDO-ORIGINAL-COM-CHAT-ATUAL.
3. Dê dois cliques somente em:

   PUBLICAR_ORIGINAL_COM_CHAT_ATUAL.bat

4. Se uma janela de login abrir, conclua o login solicitado.
5. Aguarde a mensagem "SITE ATUALIZADO COM SUCESSO".

O publicador usa o projeto Cloudflare Pages existente chamado
"vestibulando", por meio do repositório GitHub que já está conectado.
Ele não cria outro projeto.

O publicador atualiza somente as regras e os índices do Firestore.
Ele não publica regras do Firebase Storage.

VALIDAÇÕES DESTA ENTREGA
------------------------

- Build de produção concluído com sucesso.
- Arquivos visuais e páginas comparados com o ZIP original.
- 30 arquivos do chat comparados com a versão atual, sem divergências.
- JSON de índices validado.
- Pacote final testado após a compactação.
