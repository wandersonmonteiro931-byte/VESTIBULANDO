# Financeiro sem Firebase Storage

Esta versão funciona sem Cloud Storage e não exige ativação do plano Blaze apenas para comprovantes.

Fluxo:
1. O aluno copia o PIX ou abre o link de pagamento.
2. Depois de pagar, clica em **Informar pagamento**.
3. A fatura passa para **Pagamento informado**.
4. A diretoria confere o recebimento e confirma ou recusa.

O chat não foi alterado.
